# Momcozy AI 视频创作平台 — 操作手册 v1.0

> **适用版本**: Pro 版 · **更新日期**: 2026-04-30  
> **大模型矩阵**: DeepSeek-V4-Pro (文本) · GPT-4o Image (图像) · Happy Horse (视频) · CosyVoice2-0.5B (语音)

---

## 一、平台概述

Momcozy AI 视频创作平台是一个面向品牌营销团队的智能视频生产工具。输入产品信息、品牌素材和创作方向，AI 自动完成策略生成、脚本撰写、视频生成、配音配乐、质量审计——从创意到成片，全链路自动化。

**核心能力**:
- 6 种视频创作场景，覆盖直拍、品牌片、网红二创、VLOG 等需求
- Expert Studio 逐步模式：每一步可审、可改、可重做
- Smart Create 一键模式：选场景 → 填信息 → 点生成 → 等出片
- 4 个内置 AI 质量审计关口，自动拦截低质产出

**目标用户**: 品牌营销团队 · 内容运营 · 电商运营 · 创意导演

## 二、快速开始（3 步出片）

### Step 1: 选择场景
首页顶部有场景标签栏，选择你要的视频类型。每种场景的输入表单不同：

| 场景 | 适合 |
|------|------|
| **商品直拍** | 输入产品名+卖点，AI 自动生成展示视频 |
| **品牌宣传片** | 品牌形象大片，统一调性多平台 |
| **网红二创** | 已有视频素材 → 带商品链接的二次创作 |
| **实拍素材生成** | 上传实拍素材 → AI 添加旁白+字幕 |
| **品牌VLOG** | 六视图+模特+场景 → VLOG 叙事短片 |
| **快速模式** | 一句话描述 → 10-15s 快速出片 |

### Step 2: 选择操作模式

| 模式 | 特点 | 适合 |
|------|------|------|
| **Smart Create** | 全自动一键生成，AI 做全部决策 | 快速验证、批量生产 |
| **Expert Studio** | 逐步执行，每步可查看/编辑/重做 | 精细化控制、品牌审核 |

### Step 3: 生成并查看
- 点击「开始生成」后状态栏实时展示进度
- 生成完成后进入结果页：可查看策略、脚本、视频、缩略图
- 支持下载 JSON 完整数据或单独导出视频文件

## 三、6 场景详解

### 3.1 商品直拍 (S1)

**输入**:
- 产品名称 / 品牌名
- 核心卖点（每行一个）
- 品牌语气规范（可展开高级设置）

**AI 流程**: 策略生成 → 脚本撰写 → 合规检查 → 分镜设计 → 视频提示词 → AI 视频片段 → TTS 配音 → 缩略图 → 合成最终成片 → 质量审计

**输出**: 策略简报、视频脚本、AI 生成视频片段、配音音频、封面缩略图、最终合成 MP4

### 3.2 品牌宣传片 (S2)

**输入**: 品牌规范包、活动主题、关键信息、目标受众

**流程**: 同 S1，额外包含品牌合规检查步骤

**输出**: 多平台适配内容（Shopify/Amazon/TikTok）

### 3.3 网红二创 (S3)

**输入**: 已有视频 URL、产品名称、网红名称（可选）

**流程**: 原视频分析 → 重混脚本生成 → 产品链接嵌入 → 视频生成

**输出**: 含产品链接的二创视频

### 3.4 实拍素材生成 (S4)

**输入**: 实拍素材文件、产品信息、主题

**流程**: 素材分析 → 旁白生成 → 字幕同步 → 合成

**输出**: 带 AI 旁白和字幕的成片

### 3.5 品牌VLOG (S5) · NEW

**输入**:
- 品牌规范（下拉选择）
- 产品 SKU（选定后自动回填六视图）
- 场景（职场/客厅/卧室/儿童房/户外/厨房 6 选 1）
- 模特（支持多选：母亲/父亲/婴儿/孕妈/护理师/父母双人）
- 故事描述（300 字以内）
- 视频时长（5-15s / 15-30s / 30-45s / 45-60s / 60-90s）

**AI 流程**: 六视图+模特+场景+故事 → LLM 生成分镜脚本 → 每镜独立视频片段 → 配音 → 拼接

**输出**: VLOG 叙事短片，含产品六视图展示、人物使用场景、品牌收尾

### 3.6 快速模式

**输入**: 一句话描述，10-15 秒快速测试大模型矩阵能力

**流程**: 精简版 pipeline，跳过策略和分镜步骤

**输出**: 10-15s 测试短片

## 四、高级功能

### 4.1 Expert Studio 逐步模式

区别于 Smart Create 的一键自动，Expert Studio 让你在每一步之间有控制权：

- **执行此步**: 运行当前步骤
- **查看/收起**: 展开查看步骤输出
- **编辑**: 手动修改 JSON 输出数据
- **重新生成**: 丢掉当前输出重跑
- **继续执行剩余步骤**: 一键执行后续所有步骤

每个步骤完成后的状态：✅ 已完成 / 🔄 执行中 / ⏳ 待执行

### 4.2 Gate 审核关口

Expert Studio 模式下有 4 个审查节点，自动暂停等待你确认：

| 关口 | 在哪个步骤后 | 审查内容 |
|------|-----------|---------|
| Gate 1 | 脚本生成后 | 选择 1-2 个最佳脚本方向 |
| Gate 2 | 关键帧后 | 审查 AI 生成的 keyframe 图片 |
| Gate 3 | 视频片段后 | 选择满意的视频片段 |
| Gate 4 | 成片后 | 最终审查并展示多版本对比 |

每个关口可以选择「通过」→ 继续下一步，或「返回修改」→ 重做当前步骤。

### 4.3 审核面板（品牌宣传片专用）

在品牌宣传片模式下，有 4 个人工审核节点：

1. **策略审核** — 审查 AI 生成的每周内容策略
2. **脚本审核** — 逐条确认视频脚本
3. **成片初审** — 审查剪辑初稿
4. **缩略图选稿** — 选择最佳封面方案

每个审核点有三个操作按钮（按优先级排列）：
- **通过** (主按钮，绿色填充) → 继续
- **要求修改** (次级按钮，outline) → 返回重做
- **拒绝** (危险按钮，红色 outline 右置) → 终止管道

### 4.4 中止机制

在任何长等待中（策略生成、视频渲染等），点击 loading 遮罩层底部的「取消」按钮即可中断当前操作。中断后：
- 不会弹出错误提示
- Expert Studio 模式保留已完成步骤的结果
- Smart Create 模式尝试获取部分完成的步骤

## 五、管理功能

### 5.1 品牌资产 (顶栏 → 品牌资产)

管理品牌的视觉规范包：创建、编辑、上传品牌素材（Logo、色板、字体文件）、删除。

### 5.2 作品集 (顶栏 → 作品集)

浏览和管理所有已生成的视频素材文件。支持按类型筛选、预览播放、下载。

### 5.3 网红管理 (顶栏 → 网红管理)

管理合作网红/创作者的信息档案，用于 S3 网红二创场景。

## 六、设置与连接

### 6.1 API 连接配置

点击顶栏右侧齿轮图标进入设置：
- **后端地址**: 默认 `http://localhost:8001`，可修改
- **API Key**: 用于前后端认证
- **连接测试**: 点击「测试连接」按钮确认后端可用

### 6.2 演示模式

在没有后端 API 的情况下，平台自动检测并切换为演示模式（Demo Mode）。此时所有功能使用静态示例数据，可完整走通交互流程。

## 八、常见问题

**Q: 为什么视频看起来像产品旋转，没有故事线？**
A: 已通过 narrative_shot 架构修复。每个镜头现在有独立的 Shot type / Action / Camera / Scene 描述。

**Q: 生成中途可以取消吗？**
A: 可以。点击 loading 遮罩下方的「取消」按钮。

**Q: Demo 模式和真实模式有什么区别？**
A: Demo 模式使用预置示例数据，不需要后端服务或 API key。真实模式连接后端管道和 AI 服务。

**Q: 怎样优化生成质量？**
A: ① 输入更详细的产品卖点和品牌语气；② 使用 Expert Studio 模式逐步审查；③ 在 Gate 关口选择最满意的方案。
