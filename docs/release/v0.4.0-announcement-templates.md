Hi team,

AI Video Pipeline v0.4.0 has shipped to production. Highlights:

  re-login)
- Production secrets file .env.prod chmod 600
- API keys rotated (please use the new key issued by ops; old keys
  revoked)

\ud83c\udfac Capability upgrades
- S5 Brand VLOG now supports 30 / 45 / 60 / 90s multi-segment output
  (Seedance 2 + cross-segment last-frame anchoring for visual continuity)
- S2 Brand Campaign v2 (Kling 3.0 Pro 1080p, 4 cuts)
- Gate scoring now uses model-specific thresholds (less false-blocking)
- Remotion fan-out: one source \u2192 9:16 + 1:1 + 16:9 in one shot

\u2696\ufe0f Compliance fine-tuning
- 110 marketing-safe terms (\"natural\", \"organic\", \"doctor recommended\",
  etc.) no longer hard-block compliance
- Strict medical claims (treat / cure / diagnose / prevent) still blocked

\ud83d\udcca Observability
- 5 new Prometheus metrics for SLOs:
  llm_api_errors_total / llm_api_duration_seconds /
  db_pool_available_connections / admin_login_attempts_total /
  tenant_active_count
- audit_logs table captures business events (login/logout now wired,
  more in follow-ups)

\ud83d\udee1\ufe0f Resilience

  silently masquerades as real

  scheduled before EU AI Act 2026-08-02)

\ud83e\uddea Quality
- 1127 tests collected
- 119 release-critical tests 100% PASS
- 66h canary burn-in zero alerts

User-visible changes:
- Admin: re-login once (24h cookie, no further friction)
- API consumers using X-API-Key: **must use the rotated key from ops**
- Everything else: no action required

Issues post-release \u2192 oncall via Slack / DingTalk / email.
Full release notes: docs/release/v0.4.0.md on GitHub.

Cheers!
