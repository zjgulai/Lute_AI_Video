# 品牌资产收集模板 — 短视频Agent系统输入

> 按照清单逐项整理，标注【已有】或【缺失】。缺失项将直接影响对应Agent的产出质量。

---

## 1. 品牌视觉资产

### 1.1 基础视觉
- [ ] **Logo**：SVG/AI源文件 + PNG透明底（512px / 1024px / 2048px）
- [ ] **品牌色板**：主色、辅色、强调色 HEX + RGB，使用比例规则
- [ ] **字体族**：标题+正文字体名称、字重、使用场景
- [ ] **品牌视觉规范文档**（如已有PDF品牌书）

### 1.2 短视频视觉扩展
- [ ] **字幕样式偏好**：字体/颜色/描边/动画风格（弹跳/渐显/打字机）
- [ ] **信息图/贴纸风格**：圆角/直角、图标风格（线性/面性）
- [ ] **转场偏好**：硬切/叠化/滑动/Zoom
- [ ] **文字动效**：花字弹跳 / 简洁字幕 / 大标题冲击

---

## 2. 品牌语调资产（Tone of Voice）

### 2.1 品牌人格
- [ ] **品牌原型**：Caregiver / Sage / Explorer / Everyman / Hero / 其他
- [ ] **语调关键词**（3-5个，如 Warm, Empowering, Professional, Real）
- [ ] **"我们这样说话"示例**：3个最能代表品牌的真实文案
- [ ] **"我们绝不这样说话"反例**：3个

### 2.2 受众画像
- [ ] **核心用户**：年龄/身份/痛点/使用场景（例：28岁职场妈妈，泵奶碎片化，需静音免手扶方案）
- [ ] **用户原话**：用户如何描述自己的问题（如"I need to pump without being chained to a wall"）
- [ ] **搜索关键词**：用户会搜什么（"wearable breast pump"、"hands-free pumping"、"quiet pump for the office"）

---

## 3. 产品信息资产

- [ ] **产品名+型号**：全SKU清单
- [ ] **USP/核心卖点**：逐条标注优先级 P0/P1/P2
- [ ] **技术参数表**：重量/续航/噪音/材质/容量
- [ ] **认证/奖项**：FDA / CE / 红点 / 母婴奖项
- [ ] **FAQ文档**：客服高频问题 Top 20
- [ ] **竞品对比矩阵**：Momcozy / Elvie / Willow / Spectra 等，标注差异化优势

---

## 4. 素材资产

### 4.1 自有素材
- [ ] **产品白底图**：多角度（正/侧/45°/细节）
- [ ] **场景图**：办公室/家中/出行/夜间使用
- [ ] **操作视频**：佩戴/拆卸/清洗/充电流程
- [ ] **3D模型**：GLB/USDZ/STEP 格式（如有）
- [ ] **UGC授权素材库**：标注授权范围+期限

### 4.2 素材元数据标准
```
文件名 | 类型(产品图/场景/B-roll) | 角度 | 授权状态 | 有效期 | 标签
```

---

## 5. 合规与平台政策

- [ ] **Meta广告政策**：母乳喂养相关限制条款摘录
- [ ] **TikTok社区准则**：身体裸露/医疗声明限制
- [ ] **YouTube内容政策**：医疗健康内容声明要求
- [ ] **已知敏感词/限流词清单**（各平台）
- [ ] **可做声明清单**（有证据支撑的：如"Hospital-grade suction"、"FDA cleared"）
- [ ] **禁止声明清单**（夸大健康效益等）
- [ ] **免责声明模板**：视频末尾标准文案

---

## 6. 历史数据资产

- [ ] **历史爆款Top 20**：链接+播放量+完播率+转化
- [ ] **历史翻车Bottom 10**：链接+问题分析
- [ ] **各平台账号数据**：粉丝量/平均互动率/最佳发布时间
- [ ] **高转化视频共性总结**（如有）

---

## 收集优先级

| 优先级 | 类别 | 影响Agent |
|--------|------|-----------|
| **P0** | 产品USP + 禁忌清单 | 脚本、合规 |
| **P0** | 合规红线 | 合规（封号级） |
| **P0** | Logo+品牌色 | 封面、字幕、剪辑 |
| **P1** | 品牌语调 | 脚本、本地化 |
| **P1** | 自有素材库 | 素材采编、剪辑 |
| **P2** | 历史数据 | 策略优化 |
| **P2** | 3D模型 | 增强素材（非必需） |
